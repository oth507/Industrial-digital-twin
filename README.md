# Industrial Digital Twin & Predictive Maintenance System

## Overview
This project implements an Industry 4.0 Cyber-Physical System (CPS) designed to optimize energy consumption and predict equipment failures in manufacturing facilities. 

## Key Outcomes (Case Study)
* **Baseline daily energy consumption:** 480 kWh
* **Optimized daily consumption:** 402 kWh
* **Energy reduction achieved:** 16%
* **Annual impact:** ~28,000 kWh saved and ~12 tons of CO2 reduced per year.

## Smart Factory Architecture
* Layer 4: Enterprise & Application (Streamlit Dashboard)
* Layer 3: Cloud & Digital Twin (Simulation & AI Models)
* Layer 2: Edge/Fog Computing (Real-Time Analytics)
* Layer 1: Physical Asset & IoT (Sensors & Machines)

## Setup & Installation
1. Clone the repository.
2. Install dependencies: `pip install -r requirements.txt`
3. Run the dashboard: `streamlit run dashboard/digital_twin_dashboard.py`
